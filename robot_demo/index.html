Embodied RobAI: Advancing Embodied AI through Vision-Language Actions

The Embodied RobAI project is a bold initiative to push the frontier of Embodied AI where robots act in the physical world via sensors, perception, and motor control. This project develops Vision-Language Actions (VLA) to unify visual understanding, language, and robotics.

System Architecture

  • Embodied Visual Perception: Real-time processing of egocentric input
  • Language-Guided Decision Making: Foundation models interpret and plan
  • Action Controller: Actuation trained via reinforcement/imitation learning
  • Embodied Memory Module: Temporal-spatial awareness
