Embodied RobAI: Advancing Embodied AI through Vision-Language Actions

NYU AIR Lab

New York University, NYU Abu Dhabi

Project Summary

The Embodied RobAI project pushes the frontier of Embodied Artificial Intelligenceâ€”developing general-purpose robots through Vision-Language Actions (VLA) that unite perception, language, and control. By embedding intelligence into robots that perceive, interpret, and act in the real world, the project aims to build lifelike agents capable of real-time, goal-directed behavior.

Why Embodied AI?

Embodied AI integrates sensors, cognition, and motion into a feedback loop, enabling robots to learn from physical interaction. This fosters commonsense reasoning, spatial awareness, and adaptive autonomy beyond what virtual agents can achieve.

Core Objectives

  • Unifying VLA and Embodied AI: Build agents that see, understand, and act.
  • Learning from Interaction: Acquire new skills through visual-linguistic demonstration.
  • Scalable Generalization: Generalize to new tasks and scenes from foundation models.
  • Grounded Instruction Following: React contextually to spoken commands and surroundings.

System Architecture

  • Embodied Visual Perception: First-person vision for egocentric scene understanding.
  • Language-Guided Decision Making: VLMs guide task interpretation and planning.
  • Action Controller: Executes motion via policy networks trained from VLA demonstrations.
  • Embodied Memory Module: Maintains environmental and task state across time.

Use Cases

  • Assistive Robots for BVI and elderly
  • Home service and educational robots
  • Warehouse logistics automation
